# FraudGuard AI
Détecteur de fraudes de transactions en ligne en tant réel.

## Besoins fonctionnels

Un utilisateur doit:
[x] Visualiser les alertes récentes en live
[x] Voir les détails d'une alerte
[x] Modifier les paramètres du logiciel
[x] Marquer une nouvelle alerte comme "Fraude" ou "Faux positif"
